  • Modal icons fill/unfill when clicked

Rules

Thumbnail

  • Hover On - Thumbnail size increases
  • Hover On - White border fades in around thumbnail
  • Hover On - Play button icon fades in
  • Hover On - Info icon fades in
  • Hover Off - Thumbnail size decreases back to original state
  • Hover Off - White border fades out
  • Hover Off - Play button icon fades out
  • Hover Off - Info icon fades out

Play Button

  • Hover On - Play button size increases
  • Hover On - Cursor changes to pointer
  • Hover Off - Play button reverts back to original size
  • On Click - Play button size decreases
  • Off Click - Play button size reverts back to hover size

Info Button

  • Hover On - Info button size increases
  • Hover On - Cursor changes to pointer
  • Hover Off - Info button reverts back to original size
  • On Click - Dropdown modal fades in

Modal Buttons

  • Hover On - Background color darkens
  • Hover Off - Background color reverts to normal
  • On Click - Dropdown modal fades out
  • On Click - Modal button icon is filled in upon opening again
  • On Click - If filled in, modal button will unfill if clicked again
